## Usage Examples

### Basic Import
```tsx

```

### With TypeScript
```tsx
import { Button, type ButtonProps } from '@johnqh/mail-box-components';
```

### Dynamic Import
```tsx
const HeavyComponent = lazy(() => 
  import('@johnqh/mail-box-components').then(m => ({ default: m.HeavyComponent }))
);
```
